variable

In computer programming, a variable is a named storage location that holds a value. The value can be changed during the execution of the program.

constants

In computer programming, a constant is a named storage location that holds a value that cannot be changed during the execution of the program.
Once a constant is defined and its value is assigned, it cannot be modified later in the program. Constants are typically used to represent values that are fixed and do not change, such as mathematical constants like pi or e, or configuration settings that should not be changed during program execution.
